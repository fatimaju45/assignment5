Introduction to DOM

Some basic introductory questions about DOM

1. What is the Document Object Model (DOM)?

DOM is a programming interface that connects html with js and provides utilities to add,change or delete elements from document.

2. How do you select an element from the DOM?

There are multiple ways to get an element from DOm,but mostly we use 'getElementById'. Besides we can select element with 'getElementByClassname','..Bytagname', and also using query selector

3. What is event delegation in the context of the DOM, and why is it useful?

Event deligation is a pattern to handle events in javascript. We can manage multiple child elements at a time with it.That's why it is very useful.

4.How do you manipulate an element's attributes and styles using the DOM?

To manipulate element's attributes we use setAttributes(),getAttributes() or removeAttributes() property.
Also we use css style or classlist to manipulate element's styling.
